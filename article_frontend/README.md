# Article Viewer Frontend

A React application for viewing and optimizing articles with AI enhancements.

## Features
- Article listing with search and filter
- AI-enhanced content optimization
- Responsive design with Tailwind CSS
- Real-time article metrics

## Setup
1. Install dependencies: `npm install`
2. Start development server: `npm start`
3. Build for production: `npm run build`

## Technologies
- React 18
- Tailwind CSS
- Lucide React Icons
- Fetch API for backend communication
